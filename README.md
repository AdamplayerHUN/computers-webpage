# computers-webpage
számítógépek felépítéséről szóló weblaprendszer

A számítógép részeit kell bemutatnunk egy HTML weboldalon
Nagyjából ezekre a témákra gondoltam:
  Hardver:
    Alaplap,
    Processzor,
    Memória:
      Memóriatípusok,
    Bővítőkártyák:
      Videókártya,
      Hangkártya,
      USB-bővítőkártya,
      Modemkártya.
    Tárhelyek:
      SSD,
      HDD,
      SHDD,
      Flash,
      Optikai.
  
  Szoftver:
    Operációs rendszerek.
    Programok típusai:
      Felhasználói programok,
      Segédprogramok,
      Játékprogramok

  A számítógép működési folyamata

Projekt folyamata:
  Weblapvázlat elkészítése,
  Információk összeszedése,
  Stílus hozzáadása.
